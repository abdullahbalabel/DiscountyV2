# UI/UX Overhaul for Discounty Application This plan outlines a complete redesign of the Discounty application to make it highly premium, modern, and engaging. We will adhere to standard UI best practices, introducing a sleek new color palette, dynamic Reanimated micro-interactions, and completely restyling the core application screens and base components. ## Proposed Changes ### Core Design System - **Color Palette:** Vibrant Coral (`#FF4757`) Primary, Deep Navy (`#1E272E`) Secondary, sleek greys. - **Typography:** Larger, more impactful headers (700 weight, tighter spacing). - **Border Radius:** Modern large curves (xl: 24, squircle style). - **Shadows:** Diffused, soft "floating" drop shadows. ### Base UI Components - **Button:** Animated press states (scale to 0.97), spring physics, gradient/solid fills. - **Card:** Modernized padding/radii, animated entrance (staggered fade-up). - **AnimatedEntrance:** Reusable wrapper for slide-up and fade-in animations. ### Customer Screens & Components - **DealCard:** Full-width images, gradient overlays, floating coral discount badges, press-scale effects. - **Feed:** Redesigned header with large bold titles and sleek avatars, staggered card entrance. - **Layout:** Restyled bottom tab navigator with soft top shadows and vibrant active colors. ### Provider Screens - **Dashboard:** Modern analytics widgets with translucent backgrounds and large metrics. - **Layout:** Tab bar styling matching the customer aesthetic. ### Authentication Screens - **Onboarding/Landing:** Premium feel with animated backgrounds/large graphics and floating card login containers.
